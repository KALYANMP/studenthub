## Installation & Development

1. Clone the repository:
   ```bash
   git clone https://github.com/ComradeMohan/saveethahub.git
   ```
2. Navigate to the project directory:
   ```bash
   cd saveethahub
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the project locally:
   ```bash
   npm run dev
   ```
